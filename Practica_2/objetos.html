18. ¿Qué diferencia hay entre una propiedad y un método en un objeto?

Una propiedad es un valor o dato (numerico, candeno, boolean) que esta asociado a un objeto, mientras que los metodos es una función que esta asignada a una propiedad, ejecutandose para realizar la accion o comportamiento. 

                const persona = {
                    nombre = "Fabian",      // Esto es una propiedad
                    edad: 30,
                    llorar: function(){     // Esta es una propiedad cuyo valor es una función
                        console.log("pipipi " + this.nombre);
                    }
                }

                // Accede a la propiedad
                console.log(persona.nombre);

                // Invoca al método
                console.log(persona.llorar);


19. Menciona al menos 3 métodos básicos de los arrays (push, pop, map, etc.) y explica cómo funcionan.

push(... items)()

Agrega uno o mas elementos al final del array y devuelve la nueva longitud 

                let animaless = ["Elefante", "Jirafa", "Mono"];

                let cantidad = animales.push("vaca");
                console.log(cantidad); // Aqui devolvera 4
                console.log(animales); // Aqui: ["Elefante", "Jirafa", "Mono", "vaca"]

pop()()

Elimina el elemento de un array y lo devuelve 

                let animaless = ["Elefante", "Jirafa", "Mono"];

                let animalito = animales.push("Jirafa");
                console.log(animalito); // Aqui devolvera Jirafa
                console.log(animales); // Aqui: ["Elefante", "Mono"]

includes(element)

Devuelve true si el elemento está en el array cumple con la condición.

                    console.log("Incluye vaquita");
                }

                let animal = "Mono";

                console.log(animales.includes(animal)); // Aqui devolvera true
