6. ¿Qué diferencias hay entre var, let y const?

Var

Tiene el alcance de una sola función, solo se puede usar dentro de la función en la que se cree, puede ser reasignada y redeclarada sin problemas 

             var saludar = "Hola salon :D";
    
            function ejemplo() {
                var hola = "hola";
                //este var solo podra ser usado dentro
            }

Var

Su alcance es de un bloque, solo es accedible dentro de un código rodeado por corchetes ({}), se permite la reasignacion y no se puede redeclarar 

            let saludar = "Hola salon :D";
            let numero = 4;
    
            if (numero > 3) {
                let nombre = "Gina";
                console.log(nombre); // Se ejecuta con "Gina"
            }
            console.log(nombre);     // nombre no esta definido

Const

Las variables const mantienen valores contantes, solo se accede dentro del bloque que se declare y no puede modificarse luego de declararse 

            const holi = "Holiii :D";
            holi = "Hola mundo"; // Esto marcara error

Number

Tipos numericos

Boolean

Solo tiene los true o false. 

            let sensor = true;

            if(x){

            }

BigInt

Representacion de numeros de tamaño grande 

            let numerote = BigInt(9007199254740991);

Undefined

Representa un valor que es desconocido o aun no se inicializa

Symbol

Representa un identidicador unico e inmutable. 

            let sym1 = Symbol();

Null

Es un valor que representa la ausencia de un valor 

            let valor = null;
           console.log(valor); // Lo que se imprime es "object"

8. ¿Qué diferencia hay entre un array y un objeto en JavaScript?

Un array es una coleccion de datos ordenados del mismo tipo con los que se accede a travez de un indice, mientras que los objetos son colecciones desordenadas que se organizan por clave y valor 

            // Array
            let colores = ["Rojo", "Naranja", "Amarillo", "Verde"];

            // Objetos
            let cajaColores = {
                color: "Rojo",
                cantidad: 2,
                paquete: "Pequeño"
            };

9. ¿Qué son y cómo se usan los operadores aritméticos, lógicos y relacionales?

Los opradores aritméticos son Suma(+), Resta(-), Multiplicación(*), División(/), Módulo(%) son los que realizan los cálculos.
Los opradores lógicos son los usan expresiones para dar resultados con un booleano: AND (&&) devuelve true si las las condiciones presentadas son true, OR (||) devuelve true si cualquiera de las condiciones es verdadera, NOT(!) invierte el valor del booleano
