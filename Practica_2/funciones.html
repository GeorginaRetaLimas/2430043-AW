13. ¿Cómo se declara una función y cómo se llama en JavaScript?

Se utiliza la palabra reservada de function seguido del nombre de la funcion, parentesis con o sin los parametros, corchetes y contenido 

                function nombreFuncion(){
                    // Contenido ...
                }

                funcion suma(a, b){
                    let resultado = a + b;
                    return resultado;
                }

                let total = suma(10, 20); // Asi es como se llama a una función


14. ¿Qué significa que una función tenga parámetros y un valor de retorno?

Los parametros son las variables que una funcion puede recibir para realizar sus acciones, el valor de retorno es el contenido que va despues de la palabra reservada return es usualmente el resultado o lo generado por la función. 

                // Funcion con parametros y sin retorno
                function saludar (nombre){
                    console.log ("Holi " + nombre);
                } 

                // Función con parametros y return
                function saludarLog (nombre){
                    let saludo = "Holi " + nombre;
                    return saludo;
                } 
                console.log (saludarLog("Gina"));


15. ¿Qué son las funciones flecha (=>) y qué ventajas tienen?

Las funciones flechas son una sintaxis básica a la hora de definir funciones para que el código sea más sencillo y legible. Ofrece ventajas como mejor sintaxis retorno implicito, sin embargo no es adecuada para todos los casos. 

                // Función simple de una sola linea
                let sumar = (a, b) => a + b;

                // Función de más de una línea
                let sumar = (a, b) => {let resultado, a + b, return resultado};
