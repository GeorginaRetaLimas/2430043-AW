Introducción y contexto

1. ¿Qué es JavaScript y cuáles son sus principales usos en la web?

JavaScript o JS es un lenguaje de programacion que se utiliza en conjunto con HTML y CSS con el fin de hacer páginas web interactivas. Anteriormente las páginas eran estáticas y es gracias a JS que las páginas pueden empezar a ser dinámicas, entre sus usos:
🌟Responder a la interaccion de usurios

  • Creación de menus desplegables con click
  • Funciones especiales con clicks
  • Hacer click en una imagen para cambiarla

🌟Cambiar la distribución del contenido de la pagina web
  • Mostrar un carrusel de imagenes
  • Mantener elementos en pantalla


2. ¿En qué año y con qué propósito se creó JavaScript?

El lenguaje fue creado por Brendan Eich en 1995 para una empresa de software (Netscape), lo creo con el proposito de hacer paginas web más interactivas y dinámicas, dando oportunidad de la inclusión de funciones como las validaciones de formularios o actualizaciones sin recarga.
Este tenia tres propositos principales:

  1. Interactividad: Se buscaba que las acciones de los usuarios puedan dar pie a diferents eventos como el ejemplo del menu desplegable
  2. Validación: Otro objetivo era llevar a cabo verificaciones que no requieran envios de datos para saber si estos datos eran correctos o no.
  3. Contenido dinámico: Se buscaba que las páginas puedan ser modificadas a tiempo real y personalizadas.


3. ¿Por qué JavaScript no es lo mismo que Java?

Principalmente porque son dos lenguajes de programación diferente, ademas que manejan propositos diferentes.

Diferencias Java JavaScript
Proposito Se usa para crear aplicaciones en escritorio o telefonos, sistemas o manejo de backend con fines profesionales Se usa para crear sitios web interactivos con los usuarios
Ejecucion Es un lenguaje compilado e independiente de una plataforma. Es un lenguaje interprado, se ejecuta directamente en el navegador.
Estructura El lenguaje es más fuerte y da más miedo, requiere el tipo de datos de variables. El lenguaje es dinamico y los datos se determinan hasta que el programa se ejecuta, facilitando escritura.
Codigo 
// Declaracion de tipo numerico 
 int numero = 10; 

// Declaración de tipo string 
String saludo = "Hola profe";
 
// Se infiere que es int automaticamente
var numero = 10; 

// del mismo modo con string
var saludo = "Hola profe";


4. ¿Cómo se complementa JavaScript con HTML y CSS?

Se complementan a la hora de desarrollar una página web: HTML crea la estructura de la pagina, CSS da los estilos y JavaScript le da las interacciones, dinámicas y funcionalidad.
